# Posting Page

repositório com avaliação para certificação no curso DEVStart Senai - Trilha 2, Projeto deCertificação 2: Postagem de blog

# Descrição do Projeto

O Projeto está disposto em 2 arquivos, sendo eles:

index.html => contém todo o 'esqueleto' do projeto,\
style.css => contém todo o estilo e formatações da pagina dinâmica,\
script.js => contem toda a lógica de programação e comunicação com a API e com o HTML

Dos requisitos do projeto, estão incluidas:

| Requisito                       | Implementação no código                                                                        |
| :------------------------------ | :--------------------------------------------------------------------------------------------- |
| Banner dentro do `<main>`       | `<section class="banner">` está aninhada dentro de `<main id="home">`.                         |
